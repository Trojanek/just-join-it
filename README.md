just-join-it
============
React example app.

### Running

```sh
yarn install
yarn start
```

### Side notes

You should probably disable CORS in your browser.

[Chrome extension: Moesif Orign & CORS Changer](https://chrome.google.com/webstore/detail/moesif-orign-cors-changer/digfbfaphojjndkpccljibejjbppifbc).
